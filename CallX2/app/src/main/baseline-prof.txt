# screen open + first RecyclerView scroll (the paths users feel most).

# ignores any rule it can't resolve at install time). For a fully accurate,
# device-measured profile, run the Macrobenchmark "baselineprofile" Gradle
# module + `./gradlew :baselineprofile:pixel6Api31BenchmarkAOT` from Android
# Studio (Run > Generate Baseline Profile) and it will replace/extend this
# file automatically. See notes at bottom of this file.

# ── HOW TO REGENERATE A REAL (measured) PROFILE ────────────────────────
# 1. Create a `:baselineprofile` module using AGP's Macrobenchmark template
#    (Android Studio → File → New → Module → Baseline Profile Generator).
# 2. Point it at :app, target the MainActivity → open a chat → scroll flow.
# 3. Run `./gradlew :baselineprofile:generateBaselineProfile` on a real/
#    virtual device (needs `androidx.benchmark.macro` + `androidx.test.uiautomator`).
# 4. It writes the measured profile into app/src/main/baseline-prof.txt,
#    replacing this hand-written one — keep this file as a fallback until then.
