## 1. POST /notify/reel — Repost Notification

Sends FCM push to the original reel creator when someone reposts their reel.

### Request Body
```json
{
  "type":         "repost",
  "targetUid":    "uid_of_reel_creator",
  "senderUid":    "uid_of_reposter",
  "sender_name":  "John Doe",
  "sender_photo": "https://firebasestorage.../avatar.jpg",
  "reel_id":      "reel_abc123",
  "reel_thumb":   "https://firebasestorage.../thumb.jpg",
  "caption":      "This is amazing! Must watch 🔥"
}
```

### Response
```json
{ "success": true, "messageId": "projects/sathix-97a76/messages/..." }
```

## 4. Firebase Paths Reference

| Path                                      | Value               | Description                        |
|-------------------------------------------|---------------------|------------------------------------|
| reelReposts/{reelId}/{reposterId}         | timestamp (long)    | When repost was made               |
| userReposts/{uid}/{reelId}                | timestamp (long)    | Reels this user has reposted       |
| repostCaptions/{reelId}/{reposterId}      | {caption, ...}      | Repost caption object              |
| userRepostQuota/{uid}/{yyyy-MM-dd}        | count (int)         | Daily repost count (max 50/200)    |
| repostBlocklist/{creatorUid}/{blockedUid} | boolean             | Creator's repost block list        |
| reelRepostSettings/{reelId}              | {allowReposts, ...} | Per-reel repost settings           |
| reels/{reelId}/repostCount                | int                 | Total reposts on reel              |
| reels/{reelId}/quoteCount                 | int                 | Total quote-reposts                |
| reels/{reelId}/isQuoteRepost              | boolean             | Is this reel a quote repost?       |
| reels/{reelId}/repostedFromReelId         | string              | Original reel ID (chain detection) |

## 5. Anti-Spam Rules (Client-Side)

| Rule            | Value          | Enforced By                       |
|-----------------|----------------|-----------------------------------|
| Daily quota     | 50 / 200 reposts | ReelDailyRepostQuotaHelper       |
| Chain depth     | max 1 level    | ReelRepostWorker                  |
| Rate limit      | 3 seconds      | ReelPlayerFragment                |
| Self-repost     | blocked        | ReelPlayerFragment + Worker       |
| Privacy check   | allowReposts   | ReelPrivacySettingsActivity       |
